nanocode


A Minimalistic, Open-Source and Light-weight code editor.

GitHub | Website

About

It was made by me using:

Rust, NodeJS, Tauri, CodeMirror


Features

  • Open and Save files with CTRL + O and CTRL + S.
  • A basic file explorer that you can toggle with CTRL + L
  • Syntax Highlighting and a cool theme with the CodeMirror library.
  • The app is only 8.5mb at the moment!

Download Instructions

  • Go to the website.
  • Click the "Download" button for your platform.
  • Download and run the installer file.
  • Follow the installer's instructions.
  • You now have NanoCode installed on your system!
